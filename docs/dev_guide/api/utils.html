util Module Documentation¶

This module contains generic utility functions.

Functions¶

utils.path_to_datetime(path)¶

Returns the datetime object associated with an event path.

Parameters:path (str) – The path on the filesystem matching a fault event directory. Ending in …/<date>/<time> where <date> is of the format YYYY_MM_DD and <time> is formatted hhmmss.S
Returns:A datetime object corresponding to the event timestamp embedded in the supplied path
Return type:datetime
Raises:ValueError – if the path is not of the expected format
