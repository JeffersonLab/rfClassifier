Release Notes

February 8, 2023 - v2.0.0

This is a major update to the application with a switch from supporting multiple pluggable models, to a single embedded model.

  • Dependencies updated to latest supported versions, including the bugfix in rfwtools v1.2.1 (RFFC-19).
  • Streamlined certified installation procedure and handles multiple architectures via a single script. (RFFC-18).
  • Data validation bug that produce exception on mixed sample rate has been removed by updating package dependencies to newer versions. (RFFC-17).
  • This version now only supports a single embedded model (RFFC-16, RFFC-9).
  • Latest version no longer users a requirements-dev.txt file. (RFFC-15).
  • Windows is no longer a supported platform (RFFC-12).
  • The setup-certified.bash script includes the simple documentation on how to install this application. (RFFC-10).
  • The application tests now include tests for the embedded model. (RFFC-8).

January 10, 2019 - v1.0.2

This version includes only updates to documentation, and a bug fix for building documentation.

  • Documentation updated to include clearer steps for installing model applications in to certified repository
  • Minor typo and formatting fixes
  • The Makefile github target had a bug the affected the proper copying of built documentation to the docs directory

October 17, 2019 - v1.0.1

This version includes the following updates and bug fixes.

  • The application would not successfully run when called via symlink ( RFFC-3).
  • An analysis that returns an error causes an exception if results are printed in a table format (RFFC-4).
  • Cleaned up model installation instructions (RFFC-5).
  • Added link to Models listing on the "Links" page (RFFC-6).
  • Restructured the location of release-notes and audit files to simplify SQAM procedure
  • Updated SQAM installation procedure to include updating release notes, generating audit files, and rebuilding docs Audit Trail
