base_model Module Documentation

================================
Overview
================================
This module defines the model interface expected by the rfClassifier application.  Specifically, the base_model class
enforces that two methods, describe() and analyze(), are defined.  See their documentation below.

Additional methods are provided that ease the process of parsing and validating RF fault event directories.
